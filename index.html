About

Christopher Thompson

I am a recent graduate of Cleveland State University with a Bachelor of Science in Mathematics, and I am a junior web developer in training. I became interested in web development when I built an Intranet application during an internship with The City Mission.

In back-end development, I am skilled in object oriented Java and PHP. On the front-end side, I have experience with HTML5, CSS, and Javascript.

Currently, I am working on improving and adding more functionality to Help!, a Yelp API application that I worked on for a senior programming project. This project is providing more experience with RESTful APIs while also allowing me to improve my front-end skills.
